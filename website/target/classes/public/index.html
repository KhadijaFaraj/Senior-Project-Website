ERP System Usage Patterns

Academic Research: Retail Organizations in the Kingdom of Bahrain

Start System Analysis

Abstract

This project investigates the "Usage Gap" in Enterprise Resource Planning (ERP) systems within the retail sector[cite: 19]. By utilizing a J48 Decision Tree classifier, we analyze key organizational and behavioral factors to predict and mitigate efficiency gaps.

Key Objectives

  • Identify adoption patterns across retail sectors including Pharmacy, Supermarket, and Electronics[cite: 8, 9].
  • Evaluate major ERP vendors such as SAP, Oracle, Microsoft Dynamics, and Odoo[cite: 12, 20].
  • Predict GAP versus No_Gap outcomes based on MGT_SUPPORT, TRAINING, and EXTERNAL_TOOLS[cite: 13, 14, 16, 17].
  • Provide actionable insights for roles ranging from Staff to Senior Management[cite: 10].

Model Overview

The analysis is powered by the weka.classifiers.trees.J48 algorithm. The model processes 11 primary attributes to determine if a usage gap exists within a retail environment[cite: 8, 18, 19].
